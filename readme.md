# Cafetería

La **Cafetería** es un proyecto web basado en la información proporcionada de la cafetería, la cual ofrece sus servicios, así como postres, comida etc.

Además contiene datos de contacto y ubicación.

## Tecnologías usadas
- HTML
- CSS
- JavaScript
- Sass
